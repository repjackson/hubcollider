# www.collider.cloud

## Setup
1. Clone / fork the repository
2. `cd` into the downloaded folder
3. run `meteor`
4. go to `http://localhost:3000`


## Deploy
DEPLOY_HOSTNAME=galaxy.meteor.com meteor deploy www.collider.cloud --settings settings.json
